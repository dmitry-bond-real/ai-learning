
Необходимо создать Telegram-бота, который:
- принимает текстовые сообщения от пользователя
- передает их в локальную LLM
- возвращает сгенерированный ответ обратно в Telegram

Важно: Каждое сообщение должно обрабатываться независимо, без хранения истории диалога

Вводные данные:
- API для взаимодействия с LLM уже доступно по адресу `127.0.0.1:11434/api/generate`
- telegram-бот должен быть реализован на typescript (мне он больше нравится)
- telegram-бот должен поддерживать команды выбора модели и настройки параметров генерации

Разработай план действий для создания такого бота, включая выбор инструментов и технологий, а также примерный код для реализации.

План нужно подготовить в формате *.MD файла и чтобы он был потнятен ИИ агента, который будет реализовывать бота.
